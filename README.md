# Browser Plugins

## How to Install an Extension

#### 1. Download & Unzip

Download the repository as a ZIP file. Extract the ZIP file.

#### 2. Load in Chrome

1. Open Chrome and go to `chrome://extensions`
2. Enable **Developer mode** using the toggle in the top right corner
3. Click **Load unpacked**
4. Select the folder of the extension you want to install

The extension will now appear in your extensions list and be active in the browser.

---

## Available Extensions

### textless-x-extension

Hides text on X.com (Twitter).

---

## Notes

- These extensions are loaded in unpacked mode, so they will not auto-update.
- If you update the source files, go back to `chrome://extensions` and click the refresh icon on the extension card.
- To remove an extension, click **Remove** on its card in `chrome://extensions`.
